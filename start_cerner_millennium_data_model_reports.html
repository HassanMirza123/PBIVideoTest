Overview

The Cerner Millennium Production Data Model Reports provide a view of the database objects that support the Millennium suite of solutions.

The Data Model Diagrams are Entity Relationship Diagrams that display, for each data model, the included tables and their relationships. They are stored in a common, printable PDF format.

The Reports by Table is a fully attributed HTML report that provides table, column and index information.
It contains active links to the corresponding Data Model.
